# front-end-vue
此為[back-end-api](https://github.com/JamesChiang0406/back-end-api)的前端頁面，需搭配使用。

## 專案下載以及使用
```
  1. 在終端機上輸入 git clone https://github.com/JamesChiang0406/front-end-vue.git
  2. 輸入cd進入下載的資料夾
  3. 輸入npm install 以安裝所用之套件
  4. 輸入npm run serve執行
```
